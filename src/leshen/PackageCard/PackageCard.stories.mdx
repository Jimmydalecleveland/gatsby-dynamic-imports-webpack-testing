## Package Card with custom children

When child nodes are passed to `PackageCard`, they are rendered above the main CTA at the bottom. This allows for custom additions after the main information that is commonly found in a package card. In this example the "Terms and Conditions" link has been passed.

<Preview>
  <Story name="package card with custom children">
    <Section>
      <PackageCard
        label="Best Value"
        packageData={PACKAGE_DATA}
        cta={
          <LinkButton
            to="/test"
            variant="feature"
            data-tracking={PACKAGE_CARD.BUTTON}
          >
            Learn More
          </LinkButton>
        }
      >
        <a
          href="/terms"
          style={{
            fontSize: "10px",
            fontWeight: "bold",
            textDecoration: "none",
            color: "#444",
          }}
        >
          Terms and Conditions
        </a>
      </PackageCard>
    </Section>
  </Story>
</Preview>
